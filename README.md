SGE – Sistema de Gerenciamento de Estoque

O SGE é um sistema web simples de gerenciamento de estoque desenvolvido como projeto acadêmico. A aplicação tem como objetivo demonstrar a implementação de operações básicas de CRUD (Create, Read, Update, Delete) aplicadas ao controle de produtos.

O frontend foi construído utilizando HTML, CSS e JavaScript, oferecendo uma interface direta e funcional. No backend, o sistema conta com uma API desenvolvida em Spring Boot, responsável pela lógica de negócio e comunicação com o banco de dados MySQL.

Este projeto serve como base para aprendizado de integração entre frontend e backend, além de práticas com APIs REST e persistência de dados.
